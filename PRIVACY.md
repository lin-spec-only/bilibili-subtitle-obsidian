# 隐私说明

本扩展在本地 Edge 中运行，不使用自建服务器，也不会上传字幕、笔记或账号数据。

## 扩展处理的数据

- 当前 B 站视频页的公开页面信息、字幕元数据和字幕文本，用于生成本地 Markdown 笔记。
- 当前 Edge 会话可访问的 B 站页面请求；扩展不读取、导出或保存 Cookie、密码、token、localStorage 或账号资料。
- 用户明确选择的 Obsidian 文件夹句柄，仅保存在扩展自身的 IndexedDB 中，用于在该目录创建 Markdown 文件。

## 数据保存与传输

- 最近一次提取结果仅保存在浏览器会话存储中，关闭浏览器会话后会失效。
- 保存目录句柄保存在本地浏览器中；扩展不遍历该目录，也不读取既有笔记内容。
- 字幕仅从 B 站页面及其字幕 CDN 下载，随后由用户选择写入本地 Obsidian Vault。
- 扩展不会将上述数据发送给第三方服务器。

## 权限说明

- `scripting`：在当前 B 站视频标签页读取页面已提供的字幕信息。
- `storage`：保存本地插件状态和目录句柄。
- `clipboardWrite`：仅在用户点击复制按钮时写入剪贴板。
- B 站及字幕 CDN 域名权限：读取视频信息和下载字幕文件。
