# 隐私说明

本扩展在本地 Edge 中运行。可选的语音转写由仅监听 `127.0.0.1` 的本机服务完成，不会上传字幕、笔记、音频或账号数据到第三方服务。

## 扩展处理的数据

- 当前 B 站视频页的公开页面信息、字幕元数据和字幕文本，用于生成本地 Markdown 笔记。
- 当前 Edge 会话可访问的 B 站页面请求；扩展不读取、导出或保存 Cookie、密码、token、localStorage 或账号资料。
- 视频没有字幕且用户主动点击“开始本地转写”时，扩展把当前分 P 的短期音频 CDN 地址、BV号、CID、分P和时长发送给本机 `127.0.0.1:8766` 服务。不会把 Cookie 或 B站登录凭据发送给本地服务。
- 用户明确选择的目标文件夹句柄仅保存在扩展自身的 IndexedDB 中，用于在该目录创建 Markdown 文件；旧版已经保存的 Vault 根目录和相对路径信息会继续用于兼容 Obsidian 跳转。

## 数据保存与传输

- 最近一次提取结果仅保存在浏览器会话存储中，关闭浏览器会话后会失效。
- 目标文件夹句柄保存在本地浏览器中；扩展不遍历 Vault，也不读取既有笔记内容。
- 字幕仅从 B 站页面及其字幕 CDN 下载，随后由用户选择写入本地 Obsidian Vault。
- 本地转写服务直接从 B 站音频 CDN 下载音频到 D 盘临时目录，识别结束或失败后删除临时音频；最终字幕缓存在 `D:\BilibiliASR\cache`，避免重复转写。
- Whisper 模型保存在 `D:\AI_Models`。模型首次下载会访问模型提供方，但不会上传视频音频或字幕。
- 扩展不会将上述数据发送给第三方识别服务。

## 权限说明

- `scripting`：在当前 B 站视频标签页读取页面已提供的字幕信息。
- `storage`：保存本地插件状态和目录句柄。
- `clipboardWrite`：仅在用户点击复制按钮时写入剪贴板。
- B 站及字幕 CDN 域名权限：读取视频信息和下载字幕文件。
- `http://127.0.0.1:8766/*`：仅用于连接用户主动启动的本机转写服务。
